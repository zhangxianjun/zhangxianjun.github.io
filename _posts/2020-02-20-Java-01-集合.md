## 集合

### == 和 equals
1. == 比较的是栈内存存储的内容
2. 自定义的类没有复写equals方法，等同于 ==。
3. 自定义的类复写equals方法，比较对象堆内存的数据。
4. 复写hashCode减少equals的次数，提升执行效率。
